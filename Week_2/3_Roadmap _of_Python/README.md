## ADE (Agentic Development Environment)

* Debugging Assistance
* Code Explanation

* Documentation Support

* Windsurf
* GitHub Copilot Workspace

# 🎤 Frequently Asked Interview Questions

### 1. What is Python?

Python is a high-level, interpreted, general-purpose programming language known for its readability and extensive library ecosystem.

### 2. What is FastAPI?

FastAPI is a modern Python framework used to build fast, scalable REST APIs with automatic documentation.

### 3. Difference Between Compiler and Interpreter?

A compiler translates the entire program before execution, whereas an interpreter executes code line by line.

### 4. What is Pandas Used For?

Pandas is used for data analysis, manipulation, and cleaning.

### 5. What is RAG?

Retrieval-Augmented Generation (RAG) combines document retrieval with Large Language Models to produce more accurate and context-aware responses.

### 6. What is LangChain?

LangChain is a framework for building applications powered by LLMs, supporting prompt management, memory, agents, tools, and RAG workflows.

### 7. What is a Vector Database?

A vector database stores embeddings and enables semantic similarity search for AI applications.
