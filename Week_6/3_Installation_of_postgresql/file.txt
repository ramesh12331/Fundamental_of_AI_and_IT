SQL – Structured Query Language – Tables – Rows & Columns

No Sql – Non structured query language – JSON – Collections – Documents

Mongo db, Dynamo db, Elastic Search etc...

RDMS – Relational Database Management System – Software that understand language and manages the data.

Features of Sql:
> Data Retrieval – Extract data using queries
> Data Manipulation – Add, update & Delete
> Data Definition – Create or modify tables
> Filtering & Sorting – Retrieve specific data with conditions.
> Joins – Combine data from two or more tables and retrieve.
> Functions – SUM, AVG, COUNT
> Security – Users – User roles & permissions.

SELECT name, salary from employees where department = 'HR';

SQL Categories
1. DDL – Data Definition Language – CREATE, ALTER & DROP – Structure
2. DML – Data Manipulation Language – INSERT, UPDATE & DELETE – Modify data
3. DQL – Data Query Language – SELECT – Retrieve data
4. DCL – Data Control language – GRANT & REVOKE – Managing permissions.
